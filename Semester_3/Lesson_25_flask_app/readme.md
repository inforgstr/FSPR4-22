#### Structure

```
├── app

│   ├── models.py
│   └── views.py
├── config.py
├── overview
│   └── webpage.png
├── readme.md
├── run.py
└── web
    ├── static
    │   └── css
    │       └── style.css
    └── templates
```

Note: You should rename `.env.shared` to `.env.secret`
and change ``DEBUG-MODE`` to `True or False`
